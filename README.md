# Nordic app Log conversion tool

### 描述
将Nordic app保存的log数据转换为MCU端发送的原始数据

### 软件说明
* 点击软件的转换后，如果转换成功，软件会自动创建一个文件，以保存MCU发送过来的原始数据。这个文件的名字组成是：**原始文件名字-conversion.txt**
